# plane-game
웹을 공부하면서 웹에 대해 더 이해하기 위해 배운 기술들로 만든 비행기게임입니다.

## 🛠️ 게임설명
적 비행기의 공격을 피해 먼저 상대방의 비행기를 격추시키는 게임입니다.
각 비행기의 체력은 3이며 상대 비행기의 갯수를 1-4사이로 지정할 수 있습니다.
방향키로 비행기 좌우 이동이 가능하며 스페이스바를 누르면 공격이 가능합니다.
